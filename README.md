# Python - Vetores e Processos

## 📖 Objetivo

Aplicar os conceitos de manipulação de vetores e criação de processos em Python para resolução de problemas computacionais.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Vetores;
- Processos;
- Manipulação de dados;
- Resolução de problemas utilizando programação em Python.

## 💻 Tecnologias

- Python 3

## ▶️ Como executar

1. Instale o Python 3 em seu computador.
2. Clone este repositório ou faça o download dos arquivos.
3. Execute o arquivo desejado pelo terminal:

```bash
python nome_do_arquivo.py
```
